# 🛒 Excel 自動化採購清單工具

這是一個基於 Python 的自動化工具，能將產品數據轉換為具備專業樣式的 Excel 報表。

## ✨ 功能亮點
- **動態處理**：自動偵測數據行數，支持任意數量的產品。
- **精準計算**：使用 `SUMPRODUCT` 函數，精確計算「數量 × 單價」的總和（修復了初版的邏輯錯誤）。
- **專業樣式**：自動生成全表格邊框、居中對齊及表頭格式。

## 🛠️ 技術棧
- Python 3
- Openpyxl (Excel 處理庫)
- Git (版本控制)
